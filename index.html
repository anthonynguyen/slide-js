What

  • A simple javascript slideshow system
  • Create slideshows with one HTML file

Why

  • It's tiny (644 bytes minified)
  • It's fast
  • It's free of crap
  • It's awesome.

How (users)

  • ← - previous slide
  • → - next slide
  • ↑ - previous slide
  • ↓ - next slide

  • Click on the left half of the page - previous slide
  • Click on the right half of the page - next slide

How (developers)

  1. Include this file (or minified)
  2. Style it (you can try the current theme)
  3. Add the slide class to some elements
  4. Watch the magic happen!
